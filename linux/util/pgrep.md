# 0x00. 导读

# 0x01. 简介

pgrep命令 根据名称查找进程，并显示查找到的进程id。

# 0x02. 

```
-o：仅显示找到的最小（起始）进程号；
-n：仅显示找到的最大（结束）进程号；
-l：显示进程名称；
-P：指定父进程号；
-u：指定进程的有效用户ID。
-d: 指定分隔符，默认是换行
-v: 反向匹配
-c: 仅仅打印匹配进程的数目
```

```bash
# 默认只显示 PID
$ pgrep LTTBif
3988

# 加上进程名
$ pgrep -l LTTBif
3988 LTTBif

# 匹配多个进程的时候，只显示进程号最小的那个
$ pgrep -l SkyRoadTD
97908 SkyRoadTD
98041 SkyRoadTD
$ pgrep -lo SkyRoadTD
97908 SkyRoadTD
# -n 则是显示最大的

# 输出指定父进程的子进程，如：
$ pgrep -P 2380
2381
2382
2383
2384
2385

# 指定分隔符
$ pgrep ssh -d' '   
356 574 1846 2126 2133 2903 2907 5450 5470 6928 6947 10053 14117 14121 16433 18204 18734 18953 19314 21229 21721 21957 21966 21974 22123 25976 30381 30595
```

```bash
# 严格按照字符查找
$ pgrep '^ssh$' -l
```
